В этой папке находятся решения задач по программированию, выполненные Еремейкиным Сергеем.
Все решения выполнены на языке программирования Python.
Для запуска всех программ используется интерпретатор CPython 3.4.3 в Ubuntu 15.04.
Запуск производится вызовом интерпретатора Python с передачей ему пути к файлу задачи в качестве параметра командной строки
например:
python3 easy1.py
